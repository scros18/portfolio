## 📊 Facebook Ad Specifications

### Primary Ad (1200x628px)
- **Format**: Landscape
- **Aspect Ratio**: 1.91:1
- **Best for**: Feed placements
- **File Size**: < 5MB

### Feed Ad (1200x1500px)
- **Format**: Vertical
- **Aspect Ratio**: 0.8:1
- **Best for**: Mobile feed
- **File Size**: < 5MB
